User‑Product‑Order API

A Spring Boot application with Hibernate and H2 (in-memory) database. Provides CRUD operations for Users, Products, and Orders. Uses JWT authentication and supports pagination.

Features

User registration and login (JWT based)

CRUD for Products and Orders

Pagination for listing Products / Orders

H2 in-memory database

Hibernate / JPA for persistence

How to Run

Clone the repo

Set up application.properties for H2

Run the application:

mvn spring-boot:run  


Use Postman (or any REST client) to test the APIs
